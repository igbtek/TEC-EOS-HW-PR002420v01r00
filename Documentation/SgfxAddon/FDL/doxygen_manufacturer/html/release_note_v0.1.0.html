Sigfox Protocol C Library
Version 0.1.0 ( 22-Nov-2017 )

Description

First release of the Sigfox Verified Addon.

This addon is mandatory for the Sigfox Verified Certification.

Once the device has been certified Sigfox Verified, this addon can be remove from the final firmware (to save flash space).

Compatibility with the Sigfox Library : this Addon can only be used with Sigfox Lib V2.2.2 or above.
