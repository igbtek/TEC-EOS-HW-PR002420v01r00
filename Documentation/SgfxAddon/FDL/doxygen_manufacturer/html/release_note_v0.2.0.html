Sigfox Protocol C Library
Version 0.2.0 ( 20-Feb-2018 )

Description

Sigfox Verified Addon supporting the Monarch feature

This addon is mandatory for the Sigfox Verified Certification.

Once the device has been certified Sigfox Verified, and only for END PRODUCT this addon can be remove from the final firmware (to save flash space).

Compatibility with the Sigfox Library : this Addon can only be used with Sigfox Lib V2.3.0 or above for the Monarch feature.
