Sigfox Protocol C Library
Version 2.1.0 ( 19-Sep-2017 )

Description
Integrate the Payload Encryption feature

This feature allows the encryption of the payload ( with or without a Secure Element - SE ).
There are new APIs that needs to be implemented for this version:
  • MANUF_API_get_payload_encryption_flag
  • MANUF_API_aes_128_cbc_encrypt_with_key
