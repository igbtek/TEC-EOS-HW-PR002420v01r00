Sigfox Protocol C Library
Version 2.2.2 ( 21-Nov-2017 )

Description

This version integrates the main following changes:
- Change in the prototype of the SIGFOX_API_send_outofband to allow sending RC SYNC Frame ( related to Payload Encryption ) by the user.
- Payload decryption algorithm change ( Only for Payload Encryption supported and Downlink device )
- Renaming of the Library versions and features list now present in release notes.
Updates

SLL-275Synchro Bit / Synchro Frame not described properly in rf_api.c skeleton

SLL-277Payload Encryption : allow sending of OOB RC SYNC directly from User
SLL-279Change Payload decryption algorithm on downlink frame
SLL-280Store info in NVM in test mode functions
