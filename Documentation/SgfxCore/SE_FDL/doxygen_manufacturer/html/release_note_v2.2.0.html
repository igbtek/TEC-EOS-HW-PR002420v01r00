Sigfox Protocol C Library
Version 2.2.0 ( 23-Oct-2017 )

Description

Within the Sigfox Library
- Re-architecture of the test mode embedeed in the Sigfox Library
- Error code management clean-up
- New definition of the manufacturer API : there are now RF_API, MCU_API, SE_API and REPEATER_API when appropriate.

The split into several APIs allows the device maker to easily take one RF API from a chipset maker and another MCU API from a Microprocessor provider, etc ...

Documentation
- Error Code parser tool ( HELP tab )
- Update of the Device / Repeater state machine graph ( HELP tab )
- Pseudo code of the MCU, RF, SE APIs delivered
