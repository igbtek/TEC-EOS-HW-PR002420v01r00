Sigfox Protocol C Library
Version 2.0.0 ( 7-Jul-2017 )

Description
This new version of the Sigfox Library supports Secure Element.
The Sigfox Library offers a new API to support SE which manages Sigfox Credentials as well as the authentication of Sigfox frames.
An independant feature to manage encryption of the credential has been added to this version too.

Therefore the Sigfox Library is now configurable regarding the Credential Management with either:
- Credentials encrypted CRY
- Credentials not encrypted (historic library) UNC
- Credentials in Secure Element SE

As a reminder the Sigfox Libray was already configurable regarding the Spectrum Access Management with either:
- Spectrum Access FH, DC and LBT support FDL
- Spectrum Access DC and LBT support only DL
